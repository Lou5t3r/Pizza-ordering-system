README - PIZZA ORDERING SYSTEM

APPS TO INSTALL
Make sure IDLE python is downloaded to the latest version, or you have VS code.

OVERVIEW
This is a pizza ordering system which simplifies taking orders for waiters and mitigates the problems of lost orders

HOW TO RUN
1. download the file
2. open up IDLE Python and press file.
3. Select "open" and locate the file through the folder it's downloaded at
4. Once you've selected the file, press open and it will open up the source code of the program.
5. To run it, make sure you press "F5" on your keyboard, or press the run button then press "Run module"
6. It will open another window with the executed program
